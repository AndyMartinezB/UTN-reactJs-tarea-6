# Tarea Nº6: Listado de usuarios desde una API pública.

## Descripción del Proyecto

Este proyecto corresponde a la sexta entrega del Curso de Desarrollo en React JS de
la UTN.BA. La tarea consiste en implementar una lista de usuarios utilizando el consumo de una API REST con ´fetch()´, con el objetivo de practicar manejo de estados con useState y useEffect, renderizado condicional y  manejo de errores.

## Instrucciones

Para clonar el repositorio desde tu terminal:

```bash
git clone https://github.com/AndyMartinezB/UTN-reactJs-tarea-6.git
```

Para ejecutar el proyecto (React + Vite):

```bash
npm install
npm run dev
```

Luego abrir el navegador en la URL indicada por Vite (por defecto `http://localhost:5173`).
